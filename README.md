# Installation

- Add this github repository to your Magento 2 project composer.json like that
```
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/c4boomb/sales-information-module"
        }
    ],
```

- Require module from master 
```
    "require": {
        ...
        "lev/module-sales-information": "dev-master",
    }
```

- composer update
